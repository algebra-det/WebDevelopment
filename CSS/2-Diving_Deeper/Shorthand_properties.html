Shorthand Properties

Combine value of multiple properties in a songle property (the shorthand property)

For example :

  • BORDER
    • border-width: 2px;
    • border-style: dashed | solid;
    • border-color: orange

    • Instead of this we use:

    • border: 2px dashed orange

  • MARGIN
    • margin-top: 5px;
    • margin-right: 10px;
    • margin-bottom: 5px;
    • margin-left: 10px

    • Instead of this we use:

    • margin: 5px 10px 5px 10px

    Stating top, right, bottom, left respectively

    OR

    • margin: 5px 10px; - top&bottm and left&right
    • margin: 10px - top&bottm&left&right
