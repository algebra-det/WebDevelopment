Here we will see :

  • Change the css attributes via jquery:i.e. by using .css("attribute", "value")
  • Use "this" in $(this):i.e. to make changes to the current selected element of the class

    • As there can by many elements with the same class and if we use class_name instead of "this" than the changes will be implemented for all the elements of the same class name

Note: if you want to change the attributes of an html tag like iframe,h,p,etc than we use $("class/id").attr("attribute_name", "new_value") & if we want to change the attributes of the css content regarding the html tag than we use $("class/id").css("attribute_name", "new_value")
