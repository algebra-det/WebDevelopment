Arrays :
  • To create array : new Array() OR []
    For example: new tweets = new Array
           new tweets = []

  • To append array : array_name.push("data")
    For example: tweets.push("Back To Work!")
           tweets.push("I love Coffee!")

  • To delete, rename or add_at_index : array_name.splice('data')
    • To delete : array_name.splice("index", "number_of_elements")
      For example: tweets.splice(1,2)

    • To append after a certain index : array_name.splice("index", 0, "data")
      For example: tweets.splice(1, 0, "Good Morning")

    • To change data at certain index : array_name.splice("index", 1, "change_data")
      For example: tweets.splice(1, 1, "Pack Your Lunch!")

Note:

  1. If there is only two arguments in splice:means Delete
  2. If there are three arguments in splice:
    • Second argument is 0: means Appending
    • Second argument is 1: means Changing the Data
  3. If there are four arguments in splice:means Changing/Deleting and then appending
    For example: tweets.splice(1, 1, "Changing_the_data_at_1", "New_data_appended_after_that")
